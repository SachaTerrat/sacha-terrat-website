## ⚠️ Important

- **Respecte exactement les noms de fichiers** (lettres minuscules, tirets, pas d'espaces, pas d'accents)
- Les noms exacts attendus sont écrits dans les commentaires `<!-- 📸 PHOTO ... -->` dans le code
- Si tu as des doutes, fais une recherche dans VS Code (Ctrl+Shift+F) sur le nom de fichier pour voir où il est utilisé
- **Avant chaque photo**, passe par https://squoosh.app pour compresser. Cible : 100-300 Ko par image (au-delà c'est trop lourd)

## Si une photo manque

Pas grave — le site continue de fonctionner. Tu verras un dégradé violet/fuchsia à la place (c'est le "fallback" prévu dans le CSS). Tu peux ajouter les photos progressivement.
